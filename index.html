TML DASHBOARD


Rails 6 | React | Redux | React-Router | React Datepicker | MomentJS
This was a project I was building for TML Information Services. It pipes in status data from the 50 states and territories regarding MOVr (Motor Vehical Data) for background checks. It informs both employees and public users the current status of services. Known outages and notes can be created, edited or deleted. Deployment involved AWS serverless Lambdas and a static S3 React app. This helped to cut costs from an Elastic Beanstalk instance. This demo is on Heroku for demo purposes. Feel free to play around with it.
